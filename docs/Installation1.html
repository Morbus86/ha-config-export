📦 HA Config Export

Installationsanleitung für Home Assistant

Version 1.1.3

Was macht diese Integration?

Mit einem einzigen Button-Druck exportiert HA Config Export eine vollständige Übersicht deines Home-Assistant-Systems als Textdatei. Diese Datei wird automatisch per Telegram weitergeleitet und kann direkt in einen Claude-Chat hochgeladen werden – für Fehlersuche, Optimierungsvorschläge und neue Automationsideen.

Enthalten: Alle YAML-Konfigurationen, installierte Integrationen & Add-ons, Helfer, Dashboards und das Fehlerprotokoll.

1

HACS: Repository hinzufügen

Öffne in Home Assistant die HACS-Oberfläche. Klicke oben rechts auf das ⋮ Menü und wähle „Benutzerdefinierte Repositories".

Trage folgende URL ein:

https://github.com/Morbus86/ha-config-export

Kategorie: Integration – dann „Hinzufügen" klicken.

💡 Das Repository erscheint danach in der HACS-Integrationsliste. Suche nach „HA Config Export", klicke auf Installieren und starte Home Assistant neu.
2

Integration in HACS finden

Nach dem Hinzufügen: HACS → Integrationen → Suche nach „HA Config Export".

Klicke auf den Eintrag → „Herunterladen" → Home Assistant neu starten.

3

Integration einrichten – Zielordner

Nach dem Neustart: Einstellungen → Geräte & Dienste → Integration hinzufügen → „HA Config Export" suchen und auswählen.

Im ersten Schritt den Zielordner bestätigen. /backup/ ist bereits eingetragen – einfach OK klicken.

💡 OneDrive: Wenn OneDrive als Backup-Speicher eingerichtet ist, landet die Exportdatei automatisch dort.
4

Telegram-Benachrichtigung

Im dritten Schritt sind Bot-Token und Chat-ID bereits vorausgefüllt – einfach OK klicken. Du brauchst selbst kein Telegram.

⚠️ Die Benachrichtigung geht automatisch an den Empfänger. Du musst hier nichts eintragen.
5

Einrichtung abgeschlossen

Nach dem letzten Schritt erscheint die Erfolgsbestätigung. Die Integration ist jetzt aktiv.

6

Export starten

Unter Einstellungen → Geräte & Dienste → HA Config Export findest du den Button. Klicke auf „Drücken".

✅ Was danach passiert: Die Datei ha_export_for_claude_*.txt wird in /backup/ gespeichert, landet in OneDrive und wird per Telegram weitergeleitet.
↑

Updates

Neue Versionen erscheinen automatisch in HACS unter „Ausstehende Updates". Einfach auf Aktualisieren klicken – keine Neuinstallation nötig.
